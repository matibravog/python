OPERADORES LOGICOS

and 
    para comparar si dos valores son verdaderos.
or 
    para comparar si dos valores son falsos.
not 
    para invertir el valor booleano.

EXAMPLES:

    $ feliz = True
    $ millonario = False
    $ feliz and millonario
    $ False

    $ feliz or millonario
    $ True

    $ not millonario
    $ True
